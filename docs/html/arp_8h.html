Detailed Description

ARP layer.

Author
Flavien Lallemant

This file contains the definition of the ARP layer. It provides the function to handle ARP packets.

Note
There is a large debate on whether the ARP layer should be considered as a network layer or a data link layer. Some people even consider it as a layer 2.5. In this project, we consider it as a network layer.
